## Available comands: 

### Run dev server
````bash
yarn start
````

### Run css and js lint
````bash
yarn run lint
````

### Run scss lint
````bash
yarn run lint:scss
````

### Run js lint
````bash
yarn run lint:js
````

### Run fix js lint errors
````bash
yarn run lint:js:fix
````

### Run build
````bash
yarn run build
````

### Run build for prod
````bash
yarn run build:prod
````
